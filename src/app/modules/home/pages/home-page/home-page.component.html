FAQ

¿  Nuevo sitio  ?
Sí, sucedieron varias cosas:
→
Mi proveedor de hosting y dominio era limitado.
→
Migre a Google, pero su servicio de dominios (clásico, pudo ser peor) fué vendido, así que a los 3 meses migre a otro proveedor.
→
El sitio esta en GitHub Pages actualmente, redireccionado con DNS records a mi dominio gestionado en CloudFlare (Una maravilla de servicio, espero estar completamente con ellos en los próximos meses).
¿  Cómo se hizo  ?
Básicamente todo en Angular:
→
Toda la arquitectura de Front-End en Angular.
→
El estilizado esta en Tailwind CSS y hay una que otra extensión corriendo.
¿  Y el blog  ?
Paso a mejor vida, aunque el contenido lo tengo para moverlo, hay ideas:
→
Anteriormente estaba funcionando en WordPress. Considere Jekyll, pero tuve mas contras, sobre todo para compilar las gemas de Ruby hay que bajar un set de herramientas, el livereload es malo y no se diga la integración con Tailwind CSS.
→
Otra opción era usar Gatsby pero era pesado.
→
Al final use Angular, solo me queda hacer un generador estático para las publicaciones o hacer malabares con json.
→
El desarrollo se realiza en mis tiempos libres, por lo cual va a tomar tiempo.
